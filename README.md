# 🩺 Diabetes Prediction using Machine Learning

## Overview
A Machine Learning model that predicts the likelihood of diabetes using patient health parameters.

## Features
- Data preprocessing
- Model training
- Prediction
- Model evaluation

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Google Colab

## Dataset
Pima Indians Diabetes Dataset.

## Results
Predicts whether a patient is likely to have diabetes based on clinical measurements.
